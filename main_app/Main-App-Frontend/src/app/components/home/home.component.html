Welcome to Event Hall

A webapp to create and share game related events.

Search for Games with IGDB

Event Hall uses a subset of data from IGDB to allow users to search for games to be played/watched/etc. in theier events. Games can be browsed first either before or during during event creation for users to decide what games to include in their events.

Create Your Own Gaming-related Event

Create events using a simple format and send the resulting links to people you would like to invite. View your created events in a convenient event hub.
